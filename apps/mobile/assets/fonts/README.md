# Font Files

## Alice Font

Download the Alice font file and place it here:

1. **Download Alice-Regular.ttf** from one of these sources:
   - Google Fonts: https://fonts.google.com/specimen/Alice
   - Direct download: https://github.com/google/fonts/raw/main/ofl/alice/Alice-Regular.ttf

2. **Save the file as**: `Alice-Regular.ttf` in this directory (`apps/mobile/assets/fonts/`)

The font will be automatically loaded by the app when it starts.
